
Configurations:
- A MySQL database with database name and credentials specified in application.properties is needed.
- DML queries are provided in db/vending_dml.sql file

Setting up the project to run:
-Change/setup the above configurations as per requirement
-Make sure the database configurations and the configurations in application.properties match
-Import the project in SprintToolSuite or Eclipse

On Successful run:
-the UI supports actions for Insert Coin, Eject Coin, Select Product, Show Product Count, Show Sales Report
-some outputs are shown in GUI in json format
-the state transitions is printed in console
-the current inventory is printed in console
